# JQuery  이벤트

## 이벤트 작성 방법

- on(’click’, function(){}) 형태 : 하나의 요소에 이벤트를 여러개 넣을 수 있음(바인딩)
- click(function(){}) 형태

```html
<!DOCTYPE html>
<head>
    <script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>

</head>
<body>
    <button class="bu1">버튼1</button>
    <button class="bu2">버튼2</button>
    <script>
        //방식 1
        $('.bu1').on('click', function(){ //이벤트를 바인딩할 수 있음
            $(this).text('버튼클릭함');
        })
        //방식 2
        $('.bu2').click(function(){
            $(this).text('버튼2클릭함');
        })
    </script>
</body>
</html>
```

### 2개의 이벤트 넣기


### scroll

- 스크롤했을 때 위치값 반한
    
    ```html
    <!DOCTYPE html>
    <html lang="ko">
    <head>
        <script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>
        <style>
            body{
                height: 200vh;
                width: 200vw;
            }
            .di{
                position: fixed;
                border: 1px solid black;
            }
        </style>
    </head>
    <body>
        <div class="di">
            <h3>스크롤</h3>
            <span class="s1"></span>
        </div>
    </body>
    <script>
        $(window).scroll(function(){ //scroll:마우스스크롤 이벤트
            let left=$(this).scrollLeft(); //왼쪽위치
            let top=$(this).scrollTop(); //상단위치
            $('.s1').text(left+", "+top);
        })
    </script>
    </html>
    ```

### Array, index, each

- 배열관리 메서드
    
    ```html
    <!DOCTYPE html>
    <head>
        <script src="https://code.jquery.com/jquery-3.6.4.min.js"></script>
        <script>
            //배열관리 메서드
            $(document).ready(function(){
                let array=[{name:'kim', link:'http://naver.com'},
                            {name:'Shin', link:'http://google.com'}];
    
                $.each(array,function(index,item){ //(인덱스, 실질값)
                    let n = '';
                    n+= '<a href="'+item.link+'">';
                    n+='<h2>'+item.name+'</h2>';
                    n+='</a>';
                    document.body.innerHTML+=n;
                })
    
            })
    
        </script>
    </head>
    <body>
    
    </body>
    </html>
    ```

### 입력수 초과 메세지 띄우기

```html
<!DOCTYPE html>
<head>
    <script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>
</head>
<style>
    textarea{
        width: 100px;

    }

<body>
    <div class="di">
        <textarea id="t1" maxlength="100"></textarea>
        <div class="di2">
            <p class="msg1">입력수 초과됨</p>
            <p class="count"><span class="count2">0</span>/100</p>
        </div>
    </div>
    <script>
        $('#t1').keyup(function(){
            let n = $(this).val().length;
            $('.count2').text(n);
            //100글자가 넘어가면 입력수 초과됐다고 띄우기
            //red색으로 border설정
```
